A Pythagoras-tétel

a² + b² = c²

ahol a és b a befogók, c pedig az átfogó egy derékszögű háromszögben.

Gyakorlati Példák

Klasszikus 3-4-5 Háromszög

3² + 4² = 9 + 16 = 25 = 5²

Ez az egyik legismertebb Pythagoras-i triplet, amelyet már az ókori egyiptomiak is használtak.

5-12-13 Háromszög

5² + 12² = 25 + 144 = 169 = 13²

Egy további gyakori triplet, amely a gyakorlati alkalmazásokban is előfordul.

8-15-17 Háromszög

8² + 15² = 64 + 225 = 289 = 17²

A tétel alkalmazásának egy másik szép példája a geometriában.
